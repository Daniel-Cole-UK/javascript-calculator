Free Code Camp Front End Development Projects - Build a JavaScript Calculator

Task: Create a web page containing an interactive calculator.

User Stories:

- As a user, I can add, subtract, multiply and divide two numbers.
- As a user, I can clear the input field with a clear button.
- As a user, I can keep chaining mathematical operations together until I hit the equal button, and the calculator will tell me the correct output.

Created Using:

- HTML5
- Javascript
- jQuery
- CSS3
- Modernizr
- Google fonts

Future Improvements:

- Improve memory function to allow user to cycle back through operations
